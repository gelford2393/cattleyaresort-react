# Dark Mode Toggle Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a sun/moon icon toggle to the app header that switches between light and dark mode, persisted in `localStorage`.

**Architecture:** `MainLayout` holds a single `isDark` boolean initialized from `localStorage`. A `useEffect` syncs the `.dark` class on `document.documentElement` and writes to `localStorage` on every change. The toggle button sits in the header between the app title and the logout button.

**Tech Stack:** React 18, Tailwind CSS (`.dark` class variant), Lucide React (`Sun`, `Moon`), `localStorage`

- [ ] **Step 1: Add `Moon` and `Sun` to the lucide-react import**

In `src/layouts/MainLayout.tsx`, find the existing lucide-react import and add `Moon` and `Sun`:

```tsx
} from 'lucide-react';
```

- [ ] **Step 2: Add `isDark` state and sync effect inside `MainLayout`**

Add these two hooks directly after the existing `useState` and `useNavigate` calls at the top of the `MainLayout` function body:

```tsx
const [isDark, setIsDark] = useState(() => localStorage.getItem('theme') === 'dark');

useEffect(() => {
  document.documentElement.classList.toggle('dark', isDark);
  localStorage.setItem('theme', isDark ? 'dark' : 'light');
}, [isDark]);
```
